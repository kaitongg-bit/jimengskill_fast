这条视频不是单场景，而是“小猫拟人 Vlog 蒙太奇”：

1. 小猫在浴室洗漱/吹毛，像早晨出门前准备。
2. 煎锅里做吐司早餐。
3. 小猫近景吃汉堡/早餐，画面可爱生活化。

5. 小猫和车厢里的女孩互动，保留真实路人/公共交通环境。
6. 小猫用爪子玩掌机游戏，拟人化强。
7. 小猫躺在蓝色车厢座椅上睡觉。

## 核心价值

这个视频适合提炼成一个“萌宠拟人旅行 Vlog 蒙太奇生成”Skill：

- 主角是一只小猫，被包装成会生活、会出门、会坐车、会玩游戏的 Vlog 主角。
- 结构是“出门前准备 → 路上/交通 → 消遣/互动 → 休息/表情收尾”。
- 画面风格像手机记录，不是广告大片。
- 重点不是某个单一动作，而是用多个生活片段组成“猫咪带你看世界第一站”。

## 推荐即梦工具链

如果只生成 5-15 秒单条蒙太奇：

```text
generate_form_for_info_collection → text2video
```

如果用户提供猫咪照片、参考动作或参考视频：

```text
generate_form_for_info_collection → multi_modal2video
```

如果用户想做 30 秒以上、多段更稳定的 Vlog：

```text
generate_form_for_info_collection → 分镜拆解 → 多段 text2video / multi_modal2video → video_editor
```

## 建议表单字段

| 字段 | 说明 |
| --- | --- |
| 萌宠主角 | 猫咪品种、颜色、性格、是否参考上传图片 |
| Vlog 主题 | 咪咪带你看世界、城市第一站、周末出游、通勤日记等 |
| 站点/目的地 | 地铁、商场、公园、海边、便利店、游乐园等 |
| 片段数量 | 单场景 / 3 段蒙太奇 / 5 段蒙太奇 |
| 必选片段 | 起床洗漱、做早餐、坐车、探店、玩游戏、睡觉、表情收尾 |
| 镜头风格 | 手机 Vlog、贴脸自拍、生活化近景、轻微手持 |
| 情绪氛围 | 可爱、治愈、搞笑、松弛、旅行感 |
| 视频比例 | 9:16 优先，也可 1:1/16:9 |
| 时长 | 5 秒 / 10 秒 / 15 秒，长视频需拆段 |
| 是否提供参考素材 | 有则使用多模态生成 |

```text
一只{萌宠主角}作为“{Vlog主题}”的主角，拍摄一条手机 Vlog 风格的拟人旅行蒙太奇。视频包含{片段数量}个生活片段：{片段1}、{片段2}、{片段3}、{片段4}。每个片段都像真实手机随手记录，近景或贴脸自拍构图，宠物动作拟人但保留真实动物质感，画面可爱治愈、生活化、有轻微手持感。场景围绕{站点/目的地}展开，呈现“出发去看世界”的轻松旅行感。无字幕，无水印，无多余文字，不要广告棚拍感，不要恐怖或畸形动作。
```

## 生成 Skill 时的注意事项

- 第一版“猫咪刷牙”判断过窄，应改成多片段 Vlog 蒙太奇。
- 如果只用 `text2video` 做 15 秒以内视频，片段数不要太多，建议 3-4 个。
- 如果要复刻 50 秒左右的原视频节奏，应拆成多个短片段，再用 `video_editor` 合成。
- 公开发布时保留来源链接，不照搬原作者文案。
