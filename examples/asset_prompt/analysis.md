## 核心价值

这条视频适合沉淀成一个“高质量人物与场景资产提示词生成”Skill。

它解决的问题不是单次出图，而是：

```text
简单创意描述 → 导演式场景拆解 → 资产清单 → 人物/场景/道具/色彩约束 → 详细提示词 → 可生成资产图
```

## 可复用 Prompt 结构

```text
【画面元素】
主体是谁，处于什么场景，具体有哪些建筑、道具、服饰、姿态、背景层次、前景元素、中景元素、远景元素。

【导演设计】
镜头视角、景别、构图、人物调度、空间关系、叙事瞬间、动作状态。

【光影色彩】
主色调、辅助色、光源方向、时间段、空气感、材质反光、色彩一致性约束。

【执行风格】
画面质感、细节密度、真实/电影/概念设计/资产设定图风格，是否需要多视图或细节展示。

【约束词】
禁止模糊、禁止低质量、禁止多余肢体、禁止文字水印、禁止风格跑偏、禁止色彩不一致。
```

## 生成 Skill 时的注意事项

- 不能只做“帮我写一个很长 Prompt”，必须先拆资产。
- 要输出人物资产、场景资产、关键道具、色彩约束和镜头用途。
- 要能服务后续视频生成，而不是只服务单张图。
- 用户输入很简单时，Skill 要主动补足导演式设计。
- 用户如果已有剧本/分镜，应按分镜逐段生成资产提示词。
