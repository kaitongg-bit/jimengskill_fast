# 年代感视频经验学习笔记

## 核心判断

这类爆款年代感视频的关键不是简单加“复古滤镜”，而是同时做到：

- 时代物件真实
- 第一人称代入强
- 场景有生活系统
- 镜头有人的移动感
- 声音有市井环境音
- 画质保持旧录像/胶片记忆
- 时间轴有连续叙事，而不是单张图动起来

它适合沉淀成一个独立即梦 Skill：`年代感第一人称怀旧短片生成`。

## 风格锁定词

可作为全局固定风格：

```text
90年代大陆复古怀旧，VHS录像带画质，CRT老电视扫描线，胶片颗粒噪点，泛黄褪色色调，柔焦暗角，老旧镜头划痕，低饱和复古色，真人写实，市井年代感，写实渲染，纪录片质感，生活气息浓厚，模糊回忆光影
```

声音方向：

```text
无背景音乐，保留环境同期声：老街风声、旧自行车铃声、锅铲声、木案板摩擦声、街坊低语、摊贩吆喝、饭馆碗筷声、开水翻滚声、脚步声、门轴声
```

## 第一步：生图

### 1. 怀旧场景生成公式

```text
第一人称视角 + 年代 + 地点 + 身份 + 动作 + 时代道具 + 真实光影 + 胶片质感
```

### 2. 怀旧道具生成公式

```text
身份物件 + 年代物件 + 场景物件
```

可复用道具库：

- 学校：铁皮铅笔盒、红领巾、搪瓷水杯、旧课本、木课桌、粉笔灰、绿色窗框、黑板擦。
- 家庭：热水瓶、搪瓷盆、塑料暖壶、旧挂历、墙上奖状、木门闩、老式电视、花布床单。
- 医院：铁架病床、白色床单、搪瓷药盘、旧病号服、护士帽、斑驳墙面。
- 市井：木案板、铝锅、陶罐、竹篮、旧自行车、街边摊、粗瓷碗、帆布袋。

### 3. 光影选择公式

```text
光源 + 路径 + 落点 + 环境颗粒
```

## 第二步：生视频

### 控制动作公式

```text
主体 + 动作 + 场景
```

### 人物一致性公式

```text
三视图生成标准
```

### 切换场景公式

```text
原场景视角 + 过渡方式 + 新场景 + 连贯透视
```

### 镜头控制公式

```text
镜头类型 + 镜头运动 + 镜头描述
```

### 时间戳组合公式

```text
时间戳 + 主体 + 动作 + 场景 + 镜头 + 台词/环境音
```

## 镜头提示词库

### 推

- 缓慢推近，聚焦人物表情。
- 快速推镜，制造紧张感。
- 从远景推至特写，揭示细节。
- 镜头推近，人物瞳孔放大。
- 推至眼部大特写，强化情绪。

### 拉

- 缓慢拉远，从特写至全景。
- 急速后拉，揭示大场景。
- 拉镜头，人物逐渐变小。
- 从近景拉至远景，展示环境。
- 拉出水面或镜面，实现转场。

### 摇

- 镜头横摇，扫过全景。
- 上下摇镜，打量人物全身。
- 快速摇镜，制造眩晕感。
- 慢摇跟随，展现风景。
- 摇至身后，暗示有人跟随。

### 跟

- 手持跟拍，轻微晃动。
- 侧面跟拍，突出速度感。
- 背影跟拍，增强代入感。
- 正面跟拍，人物直面镜头走来。
- 跟拍并逐渐加速，营造追逐感。

### 移

- 横向移镜，跟随人物穿过人群。
- 平稳前移，保持第一人称行走视角。
- 侧向平移，展示街道层次和人物关系。
- 穿过门框、楼道、摊位之间，制造空间连续感。
- 从室内移向室外，用光线变化完成转场。

## 生成约束

- 不要只写“复古”“怀旧”，必须写清楚年代物件和生活动作。
- 不要只写滤镜，要写 VHS、CRT、扫描线、胶片颗粒、划痕、低饱和、泛黄褪色。
- 第一人称 POV 要有镜头高度、身体代入、手部或身边道具。
- 场景必须有生活系统：有人在做事，有工具在使用，有声音在发生。
- 时间轴必须有事件推进，不能每段都是一个氛围画面。
- 台词要短、口语、带地域感或亲密关系感，避免网感文案。
- 画面要写实，不要过度精致，不要现代商业广告质感。
