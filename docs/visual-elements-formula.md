# 画面元素描写公式：从参考图提炼

这张参考图的关键价值在于：它不是简单堆形容词，而是把画面元素拆成可执行的空间和生活系统。

## 核心结构

```text
全景定位 → 主体结构 → 功能设施 → 地标视觉锚点 → 生活痕迹 → 光照 → 色彩 → 摄影质感 → 避免项
```

## 1. 全景定位

先交代整张图是什么：

- 地点
- 画幅比例
- 时代/世界观
- 地形环境
- 主体在画面中的位置
- 周围大环境

## 2. 主体结构

写场景如何组织：

- 中心区是什么
- 周围怎么展开
- 有多少主要结构
- 大小层级
- 材质和颜色
- 连接关系
- 中央区、外围区、生活区、工作区等分区

## 3. 功能设施

写这个地方如何运转：

- 火堆、灶台、水井、工坊、集市、仓库
- 食物、工具、牲畜、交通工具、武器
- 人物正在做什么
- 哪些设施说明这里不是空背景，而是能生活、生产、交易的真实场所

## 4. 地标视觉锚点

写一眼能记住的东西：

- 主帐篷
- 高塔
- 旗帜
- 残骸
- 巨树
- 特殊颜色或材质
- 可作为后续镜头识别物的标志

## 5. 生活痕迹

让画面活起来：

- 人群聚集
- 小孩奔跑
- 做饭、交易、修理、搬运
- 小路、脚印、烟尘、火光、晾晒布料
- 微小但明确的行动线索

## 6. 执行层

画面元素写完后再补：

- 光照：时间、方向、阴影、空气感
- 色彩：主色、辅助色、点缀色、色彩一致性
- 摄影：镜头、构图、颗粒、真实感
- 避免项：塑料感、游戏感、CGI 感、过度 HDR、电子滤镜、风格跑偏

## 可复用模板

```text
【画面元素】
一张{画幅比例}的{镜头/视角}画面，展现{地点/世界观}中的{主体场景}。主体位于{画面位置}，周围是{大环境}。场景整体由{主体结构}组成，中心区域是{中心焦点}，外围分布着{外围结构}，通过{连接结构}形成清晰空间层级。

场景中包含完整的生活和生产设施：{功能设施1}、{功能设施2}、{功能设施3}。人物正在{行动1}、{行动2}、{行动3}，这些动作说明这里是一个真实运转的地方。画面中有{地标1}、{地标2}作为视觉锚点，便于后续视频镜头保持识别度。地面和角落散布着{生活痕迹}，增强真实感和叙事性。

【光影色彩】
{时间段}，主光从{方向}照射，形成{阴影/空气感}。主色调为{主色}，辅助色为{辅助色}，点缀色为{点缀色}，整体保持{色彩一致性}。

【摄影质感】
{镜头}，{景别}，{构图}，画面具有{质感}，用于{资产用途}。

【约束词】
禁止{负面项1}，禁止{负面项2}，禁止{负面项3}。
```
