# 先锋 AI 人像去塑料感学习笔记

## 核心公式

```text
真人不完美细节 + 非均匀光影 + 原生相机瑕疵 + 生活化磨损 + 先锋造型/构图
```

## 皮肤真实感

必加：

```text
真实人类皮肤肌理，轻微毛孔，鼻翼泛红，眼下淡淡青黑，脸颊自然红血丝，细微泪沟，少量淡淡雀斑，唇周暗沉，T区微微出油，局部哑光皮肤，局部少量油脂反光
```

禁止：

```text
无瑕肌肤，零毛孔，完美嫩肤，通透发光皮肤，磨皮滤镜，陶瓷肌，玻璃皮
```

## 五官和发丝

```text
左右眼轻微大小不一，眉毛疏密不对称，嘴角一高一低，鼻梁轻微不对称，睫毛长短错落不齐，嘴唇边缘不规整
```

```text
碎发，毛躁飞丝，几缕乱发贴额头，发际线细碎胎毛，风吹发丝，头发局部遮挡眼睛
```

## 光影

拒绝均匀柔光。优先写：

```text
树叶阴影切割在脸上，阳光局部打亮半边脸，眼窝自然阴影，鼻底厚重阴影，下颌线阴影分层，帽子或头纱遮挡形成局部暗部，正午硬阳光，侧逆光
```

环境反光：

```text
绿植环境在脸颊形成轻微蓝绿色反光，建筑白墙形成冷白色反光，傍晚光线在面中形成暖橘色反光
```

